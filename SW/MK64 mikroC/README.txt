///////////////////////////////////////////////////////////////////////////////

- HEXIWEAR_MK64 is full featured HEXIWEAR firmware for the MK64 MCU.

///////////////////////////////////////////////////////////////////////////////

- HEXIWEAR_MK64_WiFi_3G contain WiFi and 3G functionalities in addition to HEXIWEAR_MK64 firmware.

///////////////////////////////////////////////////////////////////////////////

Both firmwares requires mikroC PRO for ARM v.5.1.0.

In order to successfully compile the examples, you need to enable case sensitivity in MikroC PRO for ARM compiler:
    1) Open the Options window by selecting Tools -> Options.
    2) Open the Output tab on the left, and select Output Settings.
    3) In Compiler options, check Case sensitive.
