1. Install Python 2.7.11

2. Upgrade PIP (follow instructions here: https://pip.pypa.io/en/stable/installing/#upgrading-pip)

2a. Make sure your setup has a PATH containining the environment variables for python binaries (e.g. add C:\Python27 to PATH and reset the system)

3. Download enum library from the link below:
https://pypi.python.org/pypi/enum34#downloads
Download file called enum34-1.1.6-py2-none-any.whl
Install it by running the following in the cmd prompt from the folder where the file is downloaded:
pip install enum34-1.1.6-py2-none-any.whl

4. Merge given 'Scripts' folder with a 'Scripts' folder in python's root folder (probably 'c:\Python27\Scripts')

5. Add the following configuration parameter into Config options field in a Debugger tab in an (existing or new) OpenOCD debugging configuration: -t k64f
